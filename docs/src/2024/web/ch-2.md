### 背景

markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者 HTML）文档。Markdown 编写的文档可以导出 HTML 格式，也可以导出 PDF 格式，甚至可以导出图片格式。

同时，它的应用也较为广泛，包括但不限于：博客、文档编写、代码注释、邮件编写、微信公众号文章编写、大模型对话等。

所以，本部分我们需要大家简要学一下 markdown。
